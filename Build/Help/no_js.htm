If you have Windows XP Service Pack 2 or Vista installed

To enable javascript:

  • Go to "Tools" => "Internet Options..." in the main menu
  • Select the "Advanced" tab
  • Scroll down to the "Security" section in the list of settings
  • Make sure that the "Allow active content to run in files on My Computer" option is turned on
  • Restart your browser to continue
  • If you are still seeing this message after following these instructions, see the "If you are using Internet Explorer" section below

If you are using Internet Explorer

To enable javascript in Internet Explorer:

  • Go to "Tools" => "Internet Options..." in the main menu
  • Select the "Security" tab
  • Press the "Custom Level..." button
  • In the dialog that appears, scroll down to the "Scripting" section
  • In the "Active Scripting" subsection select "Enable"
  • Restart your browser to continue

If you are using Netscape or Mozilla
Please refer to your browser's help section on how to enable javascript.
